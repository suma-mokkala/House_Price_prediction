# 🏠 House Price Prediction

## 📌 Project Overview

This project predicts the selling price of a house based on various features such as area, number of bedrooms, bathrooms, location, and other property characteristics using Machine Learning. The objective is to build a predictive model that helps estimate house prices accurately.

---

## 🎯 Objectives

- Predict house prices using historical housing data.
- Perform data preprocessing and feature engineering.
- Train and compare different machine learning models.
- Evaluate model performance using suitable metrics.
- Visualize data patterns and model predictions.

## 📊 Dataset

The project uses a housing dataset containing different features of houses, including:

- Lot Area
- Overall Quality
- Year Built
- Number of Bedrooms
- Number of Bathrooms
- Garage Area
- Living Area
- Sale Price (Target Variable)
